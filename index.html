About Me

Allow me to introduce myself.

Hey there! My name is Julien.

I'm a front-end web developer born and raised in Toronto, Canada, and a graduate of HackerYou's Web Development Immersive Program. I love solving puzzles and learning new skills, and entering into web development has presented me with ample opportunity for both.

In my previous career as a professional musician, I developed a tireless work ethic, a strong ability to work well on a team, and the capability to perform well under pressure and in a variety of settings. I look forward to bringing these skills and more to my web development work!

I have a Web Development Immersive Certificate from HackerYou College of Technology, and a Bachelor of Music Degree from Humber College.

Project Portfolio

Here are some recent projects I created.

Gig Essentials

React / Firebase / HTML5 / CSS3 / JavaScript / GitHub

A drummer has many individual pieces of gear to keep track of when leaving for a gig, so I built a handy app to facilitate the packing process. Choose items to add to the list, check them off when they're packed, and choose to save the list for next time!

Rock Music Recommendation Generator

JavaScript / JQuery / HTML5 / CSS3 / SCSS / GitHub

With so much music being so easily accessible these days, it can be difficult to make a decision. Luckily, you can now have that decision made for you by a cute, rock music-loving dog! Answer some questions in this quiz-style app, and you'll be provided with an album recommendation, complete with a photo and a YouTube link.

Yearly Picks of Hottest Flicks

JavaScript / JQuery / APIs / HTML5 / CSS3 / SCSS / GitHub

Looking for something to watch tonight? Using this simple app powered by the Movie Database API, simply type in a year, and get back information about the top-rated movie of that year! Easy to use, and sure to cut down that Netflix browsing time.
